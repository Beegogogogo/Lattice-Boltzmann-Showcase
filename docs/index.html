Scientific Lens

Three layers of evidence

A good validation page should make the structure explicit: analytic recovery, geometric robustness, and multiphysics response. That hierarchy helps reviewers and collaborators understand the scope quickly.

01

Analytic recovery

Taylor-Green decay, Poiseuille flow, and Gaussian transport reveal whether the numerical method reproduces known solutions with controlled error.

02

03

Coupled dynamics

Conservation, acoustic damping, and particle settling broaden the story beyond single-physics benchmark recovery.

Governing Equations

Core equations kept visible

These are broad enough to frame the gallery while staying readable on a landing page.

Weakly compressible flow

\[ \frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{u}) = 0 \] \[ \frac{\partial (\rho \mathbf{u})}{\partial t} + \nabla \cdot (\rho \mathbf{u} \otimes \mathbf{u}) = -\nabla p + \nabla \cdot \left[ \mu \left( \nabla \mathbf{u} + \nabla \mathbf{u}^{T} \right) \right] + \rho \mathbf{f} \]

Frames vortex decay, channel flow, rotated inflow tests, and mass-conservation monitoring.

Advection-diffusion transport

\[ \frac{\partial \phi}{\partial t} + \mathbf{u} \cdot \nabla \phi = D \nabla^2 \phi \]

Useful for Gaussian hill benchmarks and for explaining temperature or species verification movies.

Reference profiles and decay laws

\[ u_y(x) \propto \left[ \left(\frac{H}{2}\right)^{(n+1)/n} - \left| \frac{H}{2} - x \right|^{(n+1)/n} \right] \] \[ \mathbf{u}(x,y,t) = \mathbf{u}_0(x,y)\, \exp\!\left(-2k^2 \nu t\right) \]

Channel-flow profiles and Taylor-Green decay make it easy to display quantitative agreement directly beside the animation.

Featured Benchmark

2-D Taylor-Green Vortex

This is the first benchmark in the showcase because it is compact, classical, and quantitatively sharp: a periodic vortex field with a closed-form viscous decay law. It is an effective first check for whether the solver reproduces transient momentum transport with the expected spatial pattern and decay rate.

Case description

The 2-D Taylor-Green testcase starts from a divergence-free velocity field in a periodic box. As time advances, the vortical structure decays due to viscosity while preserving the analytic spatial form. This makes the benchmark useful for validating both the temporal dissipation rate and the phase accuracy of the computed velocity field.

For a lattice Boltzmann solver, this case is a clean verification target because there are no complex boundaries, forcing ambiguities, or geometry effects hiding the core fluid dynamics. The main question is simple: does the numerical solution decay exactly the way the Navier-Stokes reference predicts?

What this testcase verifies

01

Recovery of the vortex topology and symmetry over time.

02

Correct viscous damping rate for velocity magnitude and kinetic energy.

03

Agreement between numerical and analytical fields through an L2-type error metric.

Control equations

\[ \frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{u}) = 0 \] \[ \frac{\partial (\rho \mathbf{u})}{\partial t} + \nabla \cdot (\rho \mathbf{u} \otimes \mathbf{u}) = -\nabla p + \mu \nabla^2 \mathbf{u} \]

On this page the testcase is framed with the weakly compressible flow equations used throughout the showcase, while the benchmark target itself is the classical viscous Taylor-Green decay solution.

Analytical reference solution

\[ u(x,y,t) = U_0 \cos(kx)\sin(ky)\exp(-2\nu k^2 t) \] \[ v(x,y,t) = -U_0 \sin(kx)\cos(ky)\exp(-2\nu k^2 t) \] \[ p(x,y,t) = p_0 - \frac{\rho U_0^2}{4}\left[\cos(2kx)+\cos(2ky)\right]\exp(-4\nu k^2 t) \]

These closed-form fields give a direct target for comparison, so the animation can be paired with an explicit statement of the expected decay law instead of only a qualitative visual impression.

Workflow

From simulation output to public media

This repository is the presentation layer. Generate data in your solver environment, convert the output to a short animation or figure, and publish only the media and benchmark explanation here. 

# 1. Run the benchmark in your solver workspace
# example: Taylor-Green regression + HDF export

# 2. Convert snapshots to presentation media
ffmpeg -framerate 12 -i frame_%04d.png -pix_fmt yuv420p 2d_taylor_green.mp4

# 3. Copy the media into this repo
docs/assets/media/2d_taylor_green.mp4

1. Generate benchmark data

Run the case locally, collect HDF snapshots or plots, and verify the quantitative metric you want to present.

2. Export media

Use ParaView, Python, or ffmpeg to produce a short clip or still image with clean captions and a scalar bar if needed.

3. Publish the story

Push the media to this repo and let GitHub Pages serve a clean scientific landing page without exposing the private source tree.

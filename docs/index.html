Validation blocks

Taylor-Green family

Periodic vortex decay benchmarks for transient viscous dissipation, symmetry preservation, and field-level error checks.

Poiseuille family

Pressure-driven internal-flow tests for profile recovery, wall treatment, and dimensional consistency from planar to volumetric channels.

Driven-cavity family

Recirculating-wall-bounded flows for checking vortical structures, centerline profiles, and no-slip boundary behavior in 2D and 3D.

Project Framing

A solver page rather than a loose media wall

The main message of this project is the solver itself. The current homepage therefore starts with canonical benchmarks that are widely recognizable, quantitatively defensible, and easy to map to solver behavior. Workflow

From benchmark output to media slot

The same pipeline can be reused for all six benchmark panels now, and later for aneurysm or particle videos: run the case in the solver workspace, export visualization frames, then copy the final media into the public showcase repo. 

# 1. Run a solver benchmark in your local workspace
# 2. Export frames or a short MP4 from the post-processing step
# 3. Place the media in this repo using the case slug

docs/assets/media/2d_taylor_green.mp4
docs/assets/media/3d_poiseuille.mp4
docs/assets/media/2d_driven_cavity.mp4

1. Benchmark generation

Run the 2D or 3D benchmark, collect the reference metric, and keep the final visualization sequence consistent across cases.

2. Media conversion

Use ParaView, Python, or ffmpeg to turn solver snapshots into short clips with clean color maps and optional annotations.

3. Public publishing

Drop the media into `docs/assets/media/` and let GitHub Pages serve the updated validation gallery automatically.

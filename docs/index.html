Lattice Boltzmann Solver

Canonical benchmarks first, application media next

This site is organized around the core validation path of a lattice Boltzmann solver: Taylor-Green decay, Poiseuille flow, and lid-driven cavity flow in both 2D and 3D. The layout is intentionally modular so later sections can absorb aneurysm simulations and particle-collision videos without breaking the scientific narrative.

Project Framing

A solver page rather than a loose media wall

The main message of this project is the solver itself. The current homepage therefore starts with canonical benchmarks that are widely recognizable, quantitatively defensible, and easy to map to solver behavior. Once that foundation is clear, more complex videos such as aneurysm flow fields or particle interactions can be added as an application layer.

Application Showcase

Four application directions for results presentation

Beyond the canonical benchmarks, the solver is positioned to support application-facing demonstrations in medicine, reactive transport, crystallization, and particle-laden flows. This section frames how those results can be presented.

Medical flow

Show hemodynamic simulations with velocity fields, pressure distributions, and near-wall transport patterns in vascular geometries. The next result block is organized into three concrete showcase directions.

Showcase 01

Idealized aneurysm

Present vortex development, impingement zones, and wall-shear patterns in simplified aneurysm geometries.

Showcase 02

Blood non-Newtonian behavior

Compare shear-thinning rheology against Newtonian flow to show viscosity-dependent changes in recirculation and near-wall transport.

Showcase 03

Stent

Highlight how implanted struts alter local flow organization, streamline redistribution, and hemodynamic exposure near the vessel wall.

Chemical reaction

Present coupled flow and reaction cases through concentration contours, mixing fronts, and time-resolved evolution of reactive species.

Crystallization

Highlight nucleation and crystal-growth behavior with evolving interfaces, morphology changes, and transport-limited pattern formation.

Particle flows

Display suspended-particle transport, collision events, and multiphase trajectories to illustrate interaction-dominated flow dynamics.
